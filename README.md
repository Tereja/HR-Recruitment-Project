# HR-Recruitment-Project

## Objective
To understand and practice the end-to-end recruitment lifecycle.

## Job Role
Sales Executive

## Responsibilities
- Handle customer interactions
- Generate leads and achieve targets
- Maintain client relationships

## Skills Required
- Communication skills
- Sales knowledge
- Customer handling

## Process Followed
1. Created job description
2. Sourced candidates using LinkedIn and Naukri
3. Screened and shortlisted profiles
4. Prepared interview questions
5. Maintained candidate tracker in Excel
6. Simulated interview coordination

## Tools Used
- MS Excel
- LinkedIn
- Naukri

## Outcome
Learned sourcing, screening, and recruitment coordination.
